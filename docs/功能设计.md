### ENERGY Designer 窗口设计器
1. Designer 采用 Go LCL 组件库实现
2. 窗口仿照 Lazarus 布局
参考 Lazarus IDE 功能简化实现布局功能
3. 功能实现
- 3.1 运行，预览，打开和新建窗口
- 3.2 控件分组，控件拖拽
- 3.3 控件查看器，属性和事件动态设置
- 3.4 Go LCL代码生成

### 代码结构
#### UI
1. LCL 代码方式创建UI布局
2. 主窗口为上部分和下部分
- 2.1 上部分, 左侧: 运行工具 右侧: 组件选项卡
- 2.2 下部分, 左侧: 组件树, 组件属性 右侧: 组件设计界面
- 2.2.1 组件设计界面 包含组件树, 有一个Form窗体根节点它用于显示在组件树结构 

#### 数据载体与设计组件 DesigningComponent
- 树结构(父节点和子节点)
- 每个组件有一个独立的控制功能, 设计组件有单独的组件属性树(组件属性, 组件事件)
- 设计组件数据更新, 每个组件有自己的属性列表和事件列表, 使用反射调用函数更新设计时效果, 使用节点数据更新属性树
